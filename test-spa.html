Testing Same-Origin Detection

This is a test page to verify that DeviceView's page tracking system works correctly for same-origin iframes.

How to Test

1. Load this page in DeviceView

2. Enable comment mode (press C or click Comment button)

3. Add a comment on this Home page

4. Navigate to About or Services using the navigation above

5. The comment should disappear (it's on the Home page only)

6. Add a different comment on the new page

7. Navigate back to Home - you should see only the Home comment

About This Demo

Page Tracking Technology

This demo uses a visual fingerprinting system that identifies pages by their content.

How It Works

The system creates a unique fingerprint for each page by sampling:

• Page URL and title

• H1, H2, H3 heading text content

• First paragraph content

This fingerprint creates a unique page ID that persists across sessions.
